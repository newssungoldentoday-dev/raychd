// features

Everything IRC. Nothing extra.

⚡

Powered by InspIRCd

Built on top of InspIRCd, one of the most reliable and modular IRC servers available.

🐳

Docker Ready

Comes with a Dockerfile and docker-compose.yml — spin up your IRC server in seconds.

🔒

Self-Hosted

Your server, your rules. No third-party cloud. Full control over your data and channels.

📜

Open Source

Licensed under Apache-2.0. Fork it, modify it, and make it your own.

🖥️

Shell Scripts Included

Easy install with pkg-install and handy shortcuts via raychd-shortcuts.sh.

📦

Minimal Footprint

Lightweight and fast. Ray Chad doesn't bloat your server with unnecessary dependencies.

🔗

Webhook and Dependabot Alert Supported

Webhook and Dependabot Alert to send events to the owner.

// quick start

Up in 3 commands.

# Clone the repo
git clone https://github.com/newssungoldentoday-dev/raychd.git

# Start with Docker
cd raychd && docker-compose up -d

# Or install manually
bash pkg-install
