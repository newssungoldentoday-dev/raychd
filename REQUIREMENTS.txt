# Ray Chad IRC Server - Package Dependencies

## Core Dependencies

### Build Tools
- gcc/clang (C/C++ compiler with C++11 support)
- make
- autoconf
- automake
- pkg-config
- git

### OpenSSL Libraries
- **openssl** - OpenSSL toolkit and libraries
- **openssl-dev** - OpenSSL development headers and libraries (REQUIRED)
- **libssl-dev** - Additional SSL development libraries (Debian/Ubuntu)

### Other Libraries
- zlib (optional, for compression)

## Platform-Specific Instructions

### Termux (Android)
```bash
pkg install -y build-essential git openssl openssl-dev autoconf automake pkg-config
```

**Packages:**
- build-essential
- git
- openssl
- openssl-dev ✅
- autoconf
- automake
- pkg-config
- clang
- make

### macOS
```bash
brew install autoconf automake openssl git pkg-config
```

**Packages:**
- autoconf
- automake
- openssl ✅
- git
- pkg-config

### Ubuntu/Debian
```bash
sudo apt-get install -y build-essential autoconf automake openssl libssl-dev openssl-dev pkg-config git
```

**Packages:**
- build-essential
- autoconf
- automake
- openssl ✅
- libssl-dev
- openssl-dev ✅
- pkg-config
- git

### CentOS/RHEL/Fedora
```bash
sudo yum groupinstall -y "Development Tools"
sudo yum install -y openssl-devel autoconf automake git pkg-config
```

**Packages:**
- Development Tools (group: gcc, g++, make, etc.)
- openssl-devel ✅ (equivalent to openssl-dev)
- autoconf
- automake
- git
- pkg-config

### Arch Linux
```bash
sudo pacman -Sy --noconfirm base-devel openssl autoconf automake git pkg-config
```

**Packages:**
- base-devel (includes gcc, make, autoconf, automake)
- openssl ✅
- autoconf
- automake
- git
- pkg-config

### Alpine Linux
```bash
sudo apk add --no-cache build-base autoconf automake openssl-dev git pkg-config
```

**Packages:**
- build-base
- autoconf
- automake
- openssl-dev ✅
- git
- pkg-config

### Windows (WSL/WSL2)
Use Ubuntu/Debian instructions for WSL:
```bash
sudo apt-get install -y build-essential autoconf automake openssl libssl-dev openssl-dev pkg-config git
```

## Installation Troubleshooting

### "openssl-dev not found"
- Try `libssl-dev` on Debian/Ubuntu
- Use `openssl-devel` on CentOS/RHEL
- Use `openssl` on Arch Linux (included in base-devel)

### "pkg-config not found"
```bash
# Debian/Ubuntu
sudo apt-get install pkg-config

# CentOS/RHEL
sudo yum install pkgconfig

# Arch
sudo pacman -S pkg-config

# Termux
pkg install pkg-config
```

### Compilation Errors Related to OpenSSL
```bash
# Check OpenSSL version
openssl version -d

# Verify pkg-config can find OpenSSL
pkg-config --modversion openssl

# Force OpenSSL location (if needed during configure)
./configure --prefix=$HOME/inspircd --enable-gnutls \
    LDFLAGS="-L/path/to/openssl/lib" \
    CPPFLAGS="-I/path/to/openssl/include"
```

## Automated Installation

Use the provided `pkg-install` script which automatically handles all dependencies:

```bash
./pkg-install
```

Or via curl:
```bash
curl https://raw.githubusercontent.com/newssungoldentoday-dev/raychd/main/pkg-install | bash
```

This script will:
- ✅ Detect your OS and Linux distribution
- ✅ Install all required packages including openssl-dev
- ✅ Clone Ray Chad and InspIRCd repositories
- ✅ Build and install everything automatically
- ✅ Create convenient command shortcuts

For manual installation, see GETTING_STARTED.md
